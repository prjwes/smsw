SCHOOL MANAGEMENT SYSTEM - INSTALLATION GUIDE
==============================================

STEP 1: INSTALL XAMPP/WAMP
--------------------------
1. Download XAMPP from https://www.apachefriends.org/
2. Install XAMPP to C:\xampp (Windows) or /Applications/XAMPP (Mac)
3. Start Apache and MySQL from XAMPP Control Panel

STEP 2: SETUP PROJECT FILES
---------------------------
1. Copy the entire "school-management-system" folder to:
   - XAMPP: C:\xampp\htdocs\
   - WAMP: C:\wamp64\www\
   
2. Your project path should be:
   C:\xampp\htdocs\school-management-system\

STEP 3: CREATE DATABASE
-----------------------
1. Open your browser and go to: http://localhost/phpmyadmin
2. Click "New" to create a new database
3. Name it: school_management
4. Click "Create"
5. Select the "school_management" database
6. Click "Import" tab
7. Click "Choose File" and select: database/schema.sql
8. Click "Go" to import

STEP 4: CONFIGURE DATABASE CONNECTION
-------------------------------------
1. Open: config/database.php
2. Verify these settings:
   - DB_HOST: localhost
   - DB_USER: root
   - DB_PASS: (leave empty for XAMPP)
   - DB_NAME: school_management

STEP 5: SET FILE PERMISSIONS (Important!)
-----------------------------------------
The uploads folder needs write permissions:

Windows:
- Right-click "uploads" folder
- Properties > Security > Edit
- Give "Full Control" to "Users"

Mac/Linux:
- Open Terminal
- Navigate to project folder
- Run: chmod -R 777 uploads/

STEP 6: ACCESS THE SYSTEM
-------------------------
1. Open your browser
2. Go to: http://localhost/school-management-system
3. You should see the landing page

STEP 7: LOGIN
-------------
Default Admin Account:
- Username: admin
- Password: admin123

IMPORTANT: Change the admin password after first login!

TROUBLESHOOTING
---------------

Problem: "Connection failed" error
Solution: 
- Make sure MySQL is running in XAMPP
- Check database credentials in config/database.php
- Verify database "school_management" exists

Problem: Blank page or errors
Solution:
- Enable error reporting in PHP
- Check Apache error logs in XAMPP
- Verify all files are uploaded correctly

Problem: Can't upload files
Solution:
- Check uploads folder permissions
- Increase upload_max_filesize in php.ini
- Restart Apache after changes

Problem: Theme not working
Solution:
- Clear browser cache
- Check if JavaScript is enabled
- Verify assets/js files are loaded

SYSTEM REQUIREMENTS
-------------------
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache 2.4 or higher
- Modern web browser (Chrome, Firefox, Safari, Edge)
